# SIN EXCUSAS — React + Vite + Tailwind + Leaflet (CARTO Light)

## Requisitos
- Node 18+

## Instalación
```bash
npm install
npm run dev
```

## Rutas
- `/` Home con secciones y CTA.
- `/platform` Grid de proyectos.
- `/spots` **Mapa dinámico (Leaflet)** con CARTO Light + cards.
- `/contacto` Formulario (demo).

## Notas
- **BrandStripe** con 3 bandas: naranja/negro, transparente/blanco, bordó/naranja.
- **Fuentes**: Black Ops One (títulos) e Inter (cuerpo).
- **Mapa**: react-leaflet con **CARTO Light** (cambia a `dark_all` en `MapSpots.jsx` si querés oscuro).
